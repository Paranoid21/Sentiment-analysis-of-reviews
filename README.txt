Данный проект посвящен решению задачи "Сентимент-анализ отзывов на товары", в качестве товаров выступают телефоны. Проект был вдохновлен соревнованием на сайте kaggle (ссылка: https://www.kaggle.com/c/morecomplicatedsentiment), данные для теста взяты оттуда. 


Описание содержимого:
1. demonstration -- папка с кодом демонстрации работы классификатора, в ней лежим Python объект reviews_clf, это классификатор, обученный в clf_maker.ipynb 
2. clf_maker.ipynb -- скрипт с построением классификатора и выкладками
3. data_maker.ipynb -- парсер Яндекс.Маркета
4. MKR.pdf -- методичка по расчёту межквартильного размаха (используется в clf_maker.ipynb)
5. predict.csv -- прогнозы построенного классфикатора для test
6. test.csv -- данные для проверки качества алготритма
7. train_data.csv -- данные собранные парсером, на них будем обучать классификатор


Для более быстрого и качественного понимая сути происходящего предлагаю знакомится с материалами в следующей последовательности:
1. Изучить постановку задачи по ссылке на kaggle
2. Посмотреть файл test.csv, его так же можно найти на kaggle
3. Просмотреть data_maker.ipynb, выглянуть на страницу Яндекс.Маркета (ссылка: https://market.yandex.ru/product--smartfon-samsung-galaxy-a50-64gb/394273081/reviews?show-uid=16075284185418137152316180&track=srchlink&page=1)
4. Взглянуть на скрипт clf_maker.ipynb
5. Далее можно зайти в папку visualization, в ней рекомендую открыть demo.py и следовать инструкции, написанной в шапке
